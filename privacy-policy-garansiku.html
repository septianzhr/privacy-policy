Privacy Policy – GaransiKu Scan & Simpan Struk

Last updated: 18 Desember 2025

GaransiKu Scan & Simpan Struk (“Aplikasi”) adalah aplikasi yang membantu pengguna memindai struk pembelian, menyimpan data garansi, dan mengingat masa berlaku garansi produk. Kami menghormati privasi pengguna dan berkomitmen melindungi data pengguna sesuai dengan kebijakan Google Play.

1. Informasi yang Kami Kumpulkan

Aplikasi ini tidak mengumpulkan dan tidak menyimpan data pribadi pengguna ke server mana pun. Aplikasi tidak mengumpulkan nama, email, nomor telepon, akun, atau data identitas pribadi lainnya.

2. Pemindaian Struk (OCR)

Aplikasi menggunakan fitur pemindaian kamera untuk membaca teks pada struk pembelian. Seluruh proses pemindaian dilakukan secara lokal di perangkat pengguna.

Hasil pemindaian tidak dikirim ke server dan tidak dibagikan ke pihak mana pun.

3. Penyimpanan Data

Data garansi, hasil pemindaian struk, dan gambar struk disimpan secara lokal di perangkat pengguna menggunakan penyimpanan internal aplikasi.

Aplikasi tidak memiliki sistem akun dan tidak melakukan sinkronisasi data ke cloud atau server eksternal.

4. Izin (Permissions)

Aplikasi dapat meminta izin berikut:

  • Kamera – untuk memindai struk pembelian
  • Penyimpanan – untuk menyimpan gambar struk dan data garansi

Semua izin hanya digunakan untuk mendukung fungsi utama aplikasi.

5. Berbagi Data Dengan Pihak Ketiga

Aplikasi tidak membagikan data pengguna kepada pihak ketiga, termasuk pengiklan, penyedia layanan, atau organisasi lainnya.

6. Keamanan

Aplikasi dilengkapi dengan mekanisme keamanan untuk mencegah penggunaan tidak sah atau aplikasi hasil kloning. Kami tidak menyimpan data sensitif pengguna di server mana pun.

7. Perubahan Kebijakan Privasi

Kebijakan privasi ini dapat diperbarui sewaktu-waktu. Setiap perubahan akan ditampilkan pada halaman ini.
