Privacy Policy – Mamang Cleaner

Last updated: 5 Desember 2025

Mamang Cleaner (“Aplikasi”) adalah aplikasi pembersih perangkat yang membantu pengguna menghapus file sampah, mempercepat kinerja perangkat, dan mengoptimalkan penggunaan memori. Kami menghargai privasi pengguna dan berkomitmen untuk tidak mengumpulkan atau menyimpan data pribadi apa pun tanpa persetujuan pengguna.

1. Informasi yang Kami Kumpulkan

Aplikasi tidak mengumpulkan data pribadi pengguna seperti:

  • Nama atau identitas lainnya
  • Email atau nomor telepon
  • Foto, video, atau file pribadi
  • Informasi kontak
  • Riwayat penggunaan aplikasi

Aplikasi hanya membaca informasi dasar sistem yang diperlukan untuk menjalankan fitur pembersihan, seperti:

  • Informasi penyimpanan (jumlah ruang terpakai & ruang kosong)
  • Informasi cache aplikasi
  • Informasi file sampah yang aman dihapus

Semua informasi tersebut diproses secara lokal dan tidak dikirim keluar perangkat.

2. Izin (Permissions)

Aplikasi mungkin memerlukan beberapa izin, tergantung fitur yang digunakan:

  • Storage / File Access – digunakan untuk memindai dan menghapus file sampah yang aman, seperti cache dan folder sementara.
  • Usage Access Permission – (opsional) digunakan jika aplikasi memiliki fitur analisis penggunaan memori atau CPU.

Aplikasi tidak menggunakan izin ini untuk mengakses file pribadi atau data sensitif lainnya.

3. Penggunaan Data

Mamang Cleaner tidak mengumpulkan, menyimpan, atau menganalisis data pengguna untuk tujuan apa pun. Semua proses pembersihan dan optimasi dilakukan secara lokal di perangkat.

4. Berbagi Data Dengan Pihak Ketiga

Aplikasi tidak membagikan data pengguna kepada pihak ketiga, baik untuk analitik, pemasaran, maupun tujuan lainnya.

5. Keamanan

Kami menerapkan langkah-langkah keamanan standar untuk memastikan aplikasi aman digunakan. Karena tidak ada data pengguna yang dikumpulkan atau dikirim, risiko pelanggaran data sangat rendah.

6. Penggunaan oleh Semua Kelompok Usia

Aplikasi dapat digunakan oleh semua kalangan dan tidak mengandung konten berbahaya. Mamang Cleaner tidak mengumpulkan data dari anak-anak maupun orang dewasa.

7. Perubahan Kebijakan Privasi

Kami dapat memperbarui kebijakan ini sewaktu-waktu. Perubahan akan ditampilkan pada halaman ini bersamaan dengan tanggal pembaruan terbaru.
