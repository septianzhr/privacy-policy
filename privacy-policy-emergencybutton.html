Privacy Policy – Emergency Button Indonesia

Last updated: 17 Desember 2025

Emergency Button Indonesia (“Aplikasi”) adalah aplikasi tombol darurat yang membantu pengguna mengirim pesan SOS berisi lokasi terkini kepada kontak darurat melalui WhatsApp atau SMS. Kami menghormati privasi pengguna dan berkomitmen melindungi data pengguna sesuai kebijakan Google Play.

1. Informasi yang Kami Kumpulkan

Aplikasi ini tidak mengumpulkan dan tidak menyimpan data pribadi pengguna ke server mana pun. Aplikasi tidak mengumpulkan nama, email, akun, atau data identitas pengguna.

2. Penggunaan Lokasi

Aplikasi menggunakan izin lokasi hanya saat pengguna menekan tombol SOS. Lokasi digunakan untuk membuat tautan Google Maps yang dikirim ke kontak darurat.

Data lokasi tidak disimpan dan tidak dibagikan ke server atau pihak ketiga.

3. Penggunaan SMS dan WhatsApp

Aplikasi menggunakan izin SEND_SMS untuk mengirim pesan darurat melalui SMS apabila WhatsApp tidak tersedia atau tidak ada koneksi internet.

Pengiriman WhatsApp dilakukan melalui aplikasi WhatsApp resmi tanpa mengakses atau menyimpan isi pesan pengguna.

4. Penyimpanan Data

Nomor kontak darurat disimpan secara lokal di perangkat pengguna menggunakan penyimpanan internal aplikasi. Data ini tidak pernah dikirim ke server mana pun.

5. Izin (Permissions)

Aplikasi dapat meminta izin berikut:

  • Akses Lokasi – untuk mengirim lokasi saat SOS
  • Kirim SMS – sebagai cadangan jika WhatsApp tidak tersedia
  • Akses Internet – untuk membuka WhatsApp

Semua izin hanya digunakan untuk fungsi utama aplikasi.

6. Berbagi Data Dengan Pihak Ketiga

Aplikasi tidak membagikan data pengguna kepada pihak ketiga, termasuk pengiklan, penyedia layanan, atau organisasi lain.

7. Keamanan

Aplikasi dilindungi dengan mekanisme keamanan untuk mencegah penggunaan tidak sah atau aplikasi hasil kloning. Kami tidak menyimpan data sensitif pengguna di server mana pun.

8. Perubahan Kebijakan Privasi

Kebijakan privasi ini dapat diperbarui sewaktu-waktu. Perubahan akan ditampilkan pada halaman ini.
